ADAS Measurement and Analysis System

What is it?

ADAS M/A system is necessary for the development and testing of vision AI systems (including autonomous system(s)). It functions by logging, reprocessing, HiLs and analysis of sensor data and network data for vehicles. Based on sophisticated synchronization (minimum 12.8 ns) between data, our product meet the perfect HiLs condition without loss of data

    Necessary for using:
  • Sensor data: Camera (include stereo), Radar, Lidar, GPS, IMU, etc
  • Vehicle network data : CAN (FD), FlexRay, Automotive ethernet, etc

Advantages

  • LVDS Camera, Radar, Lidar, IMU, GPS, Automotive ethernet, FlexRay and CAN signals are synchronized exactly.
  • Timestamp precision accuracy (Minimum 12.8 ns) and hardware level time synchronization.
  • Data validation function for the lossless system.
  • All-in-one product that can process all of the above data in one piece of equipment.
  • No hardware cost.

Accelerator Design Service

What do we do?

We provide a vision algorithm for robotics and autonomous vehicles with an accelerator design for real-time operation. For our customers, we have been developing accelerator development for the following features.

Features

  • YOLO-V2/3 DNN forward accelerator.
  • Classification with NiN model forward accelerator.
  • Adaboost dectection accelerator.
  • Video compression and decompression(MPEG1/2/4,H264).
  • Semi global stereo matching(8 path) up to 2MP.
  • Calibration of distortion with stereo up to 3MP.
  • Image warping.
  • Keystone for projector.
  • Pincushion for projector.
  • Scalar.
  • Image signal processor.
  • 3-dimensional optical flow.
  • Noise reduction filter same like lossless.
  • User-specific vision algorithms.

Display Drive controller Module (DDC-M)

What do we offer?

Our DDC technology provides drive performance for high frame rate and high-resolution displays, and provides an overall solution including panels, optical technologies and video processors.

Specifications

  • HDMI input for video input up to 4K/60Hz.
  • MIPI input for video input up to 4K/60Hz.
  • LVDS output for video output up to 4K/60Hz.
  • Display drive support for various panels up to 120 frame rate.
  • EDID controller.
  • OSD menu control with keypad.
  • Contrast and brightness control.
  • Gain control.
  • The drive functions to support special panels can be custom built-in if needed.
  • Supports RGB / Laser light source control if needed .

Application

  • Display drive controller for micro dispay panel same like LCOS, DLP, oLED.
  • Display drive controller for high definition LED, TFT .
  • Industry display.
  • Pico projector.
  • VR/AR.
